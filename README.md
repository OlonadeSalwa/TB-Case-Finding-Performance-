# 📊 TB Active Case Finding Performance Dashboard

## 📌 Project Overview

The **TB Active Case Finding Performance Dashboard** is an interactive Power BI report developed to monitor the performance of community tuberculosis (TB) case finding activities across five Local Government Areas (LGAs) in Lagos State, Nigeria.

The dashboard compares actual program performance against predefined screening, presumptive, and case detection targets while highlighting treatment enrolment gaps. It provides stakeholders with actionable insights for monitoring field activities and improving TB program outcomes.

## 🎯 Project Objectives

The dashboard was developed to:

- Monitor TB screening performance across supported LGAs.
- Compare actual screening results against assigned targets.
- Evaluate presumptive case identification performance.
- Track TB case detection performance.
- Identify treatment enrolment gaps.
- Support evidence-based decision-making for TB program management.

---

## 📊 Key Insights

- Surulere recorded the highest number of individuals screened.
- Kosofe had the lowest presumptive target achievement among the five LGAs.
- Eti-Osa recorded the highest TB case detection performance, achieving 50% of its target.
- Significant enrolment gaps remain in Kosofe, Shomolu, and Surulere, indicating opportunities to improve linkage to treatment.
- Screening performance across all LGAs remained below the assigned screening target, highlighting the need to strengthen community outreach activities.
